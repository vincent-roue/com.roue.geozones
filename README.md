# GeoZones

GeoZones is a Homey Pro app that lets you create circles and polygons on a map and use them directly in Homey flows.

## Version 1.0.0 highlights

- reusable circle and polygon zones
- tags on zones
- subject types
- per-zone subject and subject type filters
- subject movement threshold
- entry and exit delays
- hysteresis on circles
- evaluation with or without subject context
- distance flow cards
- location sources page for compatible Homey devices
- import, export and configuration history

GeoZones helps you detect whether a person or device is at home, at work, at school or in any other place that matters to your automations.

Use GeoZones when you want reusable zones without creating a virtual device. The app remembers the last known zone for each subject so your flows can react when someone enters, leaves or moves between the places you define.
